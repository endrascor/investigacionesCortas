¿Qué es la Usabilidad web?

La Usabilidad web se trata de hacer una pagina web de tal manera que sea sencilla a la hora
de navegar a travez de esta y atractiva visualmente para las personas que la visitan.

Importancia de la Usabilidad web:

La Usabilidad web tiene varias ventajas que la vuelven importante y necesaria en el desarrollo de una pagina web, entre las principales ventajas se encuentran:
  • Mejora la experiencia del usuario
  • Aumenta la retención de usuarios
  • Incrementa las conversiones
  • Reduce los costos de soporte
  • Mejora la imagen de la marca

¿Qué es DCU?

DCU son las siglas en inglés de Diseño Centrado en el Usuario (User-Centered Design).
Se trata de un enfoque de diseño que se antepone a las necesidades de los usuarios mientras se desarrolla
la pagina web para que los usuarios se sientan comodos y siempre encuentren lo que necesitan.

Importancia del DCU:

El DCU es importante porque permite que los usuarios naveguen por las páginas con gran comodidad.
Además, si necesitan encontrar algo específico, este diseño facilita que lo hallen de manera rápida y sencilla.

Principios de Usabilidad:

Existen 10 Principios de la usabilidad estos son los siguientes:

  1. Visibilidad del estado del sistema
  2. Relación entre el sistema y el mundo real
  3. Control y libertad del usuario
  4. Consistencia y estándares
  5. Prevención de errores
  6. Reconocer antes que recordar
  7. Flexibilidad y eficiencia de uso
  8. Diseño estético y minimalista
  9. Ayudar a los usuarios a reconocer, diagnosticar y corregir los errores
  10. Ayuda y documentación

Visibilidad del estado del sistema

Este principio nos indica que siempre tenemos que mantener informado al usuario y enviar la informacion lo mas rapido posible

Una mala practica seria que al momento de uno envie un mesaje este no se notifique

Y una buena practica seria que si se notifica si se envio

Relación entre el sistema y el mundo real

Se deben de utilizar conceptos y imagenes que sean familiares con el usuario

Una mala practica seria un boton de buscar que tenga una flecha

Una buena practica seria que este boton de buscar contenga una lupa

Control y libertad del usuario

Siempre debe de existir una manera de revertir una accion que fue realizada por un cliente por error

Una mala practica seria un carrito de compra que no permita eliminar los productos

Y una buena practica es que si incluya la opcion de eliminar productos

Consistencia y estándares

Debe de existir una consistencia y estándares en las imagenes que se utilizen para que los usuarios siempre sepan como es que funciona
